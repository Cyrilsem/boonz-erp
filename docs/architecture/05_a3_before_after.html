01The metaphor

BEFORE

A bank without security cameras

If a row appears in machines tomorrow morning, we have no first-principles way to answer "who put it there, when, and via which code path". Some tables had ad-hoc audit (e.g., warehouse_inventory_audit_log) but the schemas, columns, and write paths varied. There was no single place to ask "show me everything that wrote to a protected table in the last hour".

  • No via_rpc column anywhere → can't tell front-door from side-door writes.
  • No standard actor + actor_role capture → forensics are guess-work.
  • Audit was opt-in per table → tables added in 2026 had none.
AFTER

A bank with one ledger and a uniformed teller

Every protected-table write — once A.4 wires it up — leaves a row in write_audit_log with a uniform shape: which table, which operation, which row, who did it, what role they had, whether they used the front door, and the full before/after row as JSON. Append-only at the policy level (RLS UPDATE/DELETE = false). Three indexes designed for the three questions we'll actually ask.

  • One ledger for all 18 protected entities.
  • One trigger function (audit_log_write) — generic, parameterized by PK column.
  • Bypass-detector index: partial index on (via_rpc, occurred_at) WHERE via_rpc = false — keeps it tiny, makes "find unauthorized writes" a millisecond query.

04The trigger function (excerpt)

CREATE OR REPLACE FUNCTION public.audit_log_write() RETURNS trigger LANGUAGE plpgsql SECURITY DEFINER AS $$ DECLARE v_pk_col text; v_via_rpc text; v_rpc_name text; v_actor uuid; ... BEGIN v_pk_col := COALESCE(TG_ARGV[0], 'id'); v_via_rpc := current_setting('app.via_rpc', true); v_rpc_name := current_setting('app.rpc_name', true); v_actor := auth.uid(); -- ... extract PK from NEW or OLD via TG_ARGV[0] ... -- ... build payload as jsonb ... INSERT INTO public.write_audit_log (table_name, operation, row_pk, actor, actor_role, via_rpc, rpc_name, payload) VALUES (TG_TABLE_NAME, TG_OP, COALESCE(v_pk_val, '?'), v_actor, v_actor_role, COALESCE(v_via_rpc = 'true', false), NULLIF(v_rpc_name, ''), v_payload); RETURN COALESCE(NEW, OLD); END; $$;
